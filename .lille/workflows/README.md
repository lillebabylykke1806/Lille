# Workflows

Step-by-step procedures for humans and agents.

Suggested workflows:

- Local development setup
- iOS TestFlight release via Codemagic
- Android internal track release
- Deploy web to production (affects Capacitor remote URL)
- Adding a new Capacitor plugin safely
- Debugging native crashes (Android `MainActivity`, RevenueCat, notifications)

Each workflow should list prerequisites, commands, verification steps, and rollback notes.
